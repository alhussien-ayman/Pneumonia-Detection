About the Model

Understanding the Traditional Machine Learning approach behind the detection system

Traditional ML Approach

Unlike deep learning methods that automatically learn feature representations, this system employs a carefully designed Traditional Machine Learning pipeline with handcrafted feature extraction, offering interpretability and efficiency.

CLAHE Enhancement Contrast Limited Adaptive Histogram Equalization for optimal lung field visibility
HOG Features Histogram of Oriented Gradients capturing structural edge and shape patterns
LBP Texture Local Binary Patterns extracting fine-grained texture descriptors from lung regions
Statistical Features Mean, std, skewness, kurtosis, and intensity histogram distributions
